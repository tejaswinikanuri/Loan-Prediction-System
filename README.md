# Loan Prediction System

## Overview

This project predicts whether a loan application will be approved based on applicant details using Machine Learning techniques.

## Features

* Data preprocessing and cleaning
* Feature engineering
* Model training and evaluation
* Loan approval prediction
* Data visualization using Matplotlib and Seaborn

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Prediction

## Results

The model was trained and evaluated using standard machine learning techniques to predict loan approval status.
